# Harry Axford - Portfolio

A personal portfolio and CV website built with Next.js, designed to showcase my skills, experience, and projects as an IT Technician and aspiring Cloud Engineer.

## Overview

This project serves as a central hub for my professional identity, featuring:
-   **Interactive CV:** A detailed, digital curriculum vitae highlighting my experience with Microsoft Intune, Active Directory, and educational IT support.
-   **Blog:** A space to share technical insights and learning journeys (e.g., Neovim, Storage).
-   **Clean Design:** Built with a focus on minimalism and readability using Tailwind CSS.

## Tech Stack

-   **Framework:** [Next.js](https://nextjs.org/) (App Router)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **Icons:** [Lucide React](https://lucide.dev/)
-   **Content:** MDX for blog posts

## Getting Started

To run this project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This application is optimized for deployment on the [Vercel Platform](https://vercel.com).
